Organic Pallete
Intro
Organic Pallete is an easy to use grocery selling website. With a user friendly interface and easy accessibility to your favourite fruits, vegetables and exotic delights, Organic Pallete is your go-to website for all of your grocery needs.
Work Process
Organic Pallete creation process follows a multi step creative approach which uses reasearch, wireframing and develop to create consistent looking ui and povides user with easy interaction as well as adhere's to the latest trends.

Research

Extensive research in the domain of ecommerce grocery website while identifying the challanges, user needs and related trends.

Wireframing
